Welcome to my Machine Learning coursework repository! This directory contains all algorithms, scripts, coursework, and major projects developed during my studies in Machine Learning at Northeastern University.

## 📁 Repository Structure

### [Final Project: Predictive Code-Switching Anticipation](./finalproject-stepbystep)
This is the master directory containing our culminating final research project. 
We built a state-of-the-art causal Multitask Learning architecture upon **mBERT** explicitly designed to predict and anticipate code-switching in bilingual dialogue across 10 diverse language pairs. 

* **Key Breakthroughs**: We mathematically mapped the limits of subword predictions for NLP Code-Switching, and demonstrated incredible zero-shot transfer capabilities evaluating completely on unseen languages.
* **Paper and Presentation**: You can find our final academic LaTeX paper (`main.tex`) and the script for our defense (`PRESENTATION_DRAFT.md`) located inside the project folder.
